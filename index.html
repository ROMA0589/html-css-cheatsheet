Mi Hoja de Trucos de HTML y CSS

Una referencia personalizada para ayudarte en el desarrollo web.

Etiquetas HTML

Etiqueta Ejemplo Descripción
<h1> <h1>Título</h1> Se usa para definir un encabezado principal en una página.
<p> <p>Texto</p> Define un párrafo en HTML.
<table> <table><tr><td>Dato</td></tr></table> Se usa para crear tablas en HTML.

Propiedades CSS

Propiedad Ejemplo Descripción
color color: red; Establece el color del texto.
font-family font-family: Arial, sans-serif; Define la familia de fuentes para el texto.
background-color background-color: #f0f0f0; Establece el color de fondo de un elemento.

Selectores CSS Comunes

Selector Ejemplo Descripción
.clase .rojo { color: red; } Selecciona todos los elementos con la clase rojo.
#id #principal { font-size: 2em; } Selecciona el elemento con el id principal.
elemento p { margin: 10px; } Selecciona todos los elementos <p>.
* * { box-sizing: border-box; } Selecciona todos los elementos de la página.
elemento, elemento h1, h2 { color: blue; } Selecciona todos los elementos <h1> y <h2>.
